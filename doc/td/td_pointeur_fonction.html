Compilation, pointeur & références

Compilation conditionnelle

Compiler le code suivant, sans puis avec les options de “warnings” 

#include <iostream>
using namespace std;

int main()
{
  unsigned int entierNonSigne;
  int entierSigne = entierNonSigne;

  cout << entierNonSigne << " " << entierSigne << endl;

  for (unsigned int iEntier = 0; iEntier < entierSigne; ++iEntier)
    cout << iEntier << endl;
}

Après exécution, corriger les “erreurs”. Ajouter finalement une directive de préprocesseur d’identificateur DEBUG permettant l’affichage ou non des différentes valeurs d’entier et compiler suivant les deux cas. S’assurer du résultat en précompilant le code source.
